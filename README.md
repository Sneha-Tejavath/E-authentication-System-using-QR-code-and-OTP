# E-authentication-System-using-QR-code-and-OTP
A secure authentication system using QR Code and OTP for validating user identity. The system generates a QR code which is scanned by the user, followed by an OTP verification to ensure secure login access. This project enhances security and avoids traditional password-based attacks.
# E-authentication-System-using-QR-code-and-OTP

## Project Overview
This project presents a secure E-Authentication System using QR Code and OTP verification for validating user identity. The system improves login security by combining QR-based authentication with One-Time Password (OTP) verification to reduce unauthorized access and password-based attacks.

## Features
- QR Code based user authentication
- OTP verification system
- Secure login validation
- Improved user identity verification
- Enhanced security against traditional password attacks

## Technologies Used
- Python
- QR Code Authentication
- OTP Verification
- Cyber Security Concepts
- Authentication System Design

## System Workflow
1. User enters login credentials
2. System generates a QR Code
3. User scans the QR Code
4. OTP is generated and sent for verification
5. User enters OTP
6. Access is granted after successful validation
